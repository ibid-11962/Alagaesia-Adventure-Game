DIRECTIONS & COMMANDS

-Use the suggested commands and directions below to complete tasks. The list is not comprehensive, so in places where none of the listed commands will work, read the surrounding descriptions carefully. If you get stuck, try different commands or ask for a hint.

-Use directional commands (north, south, etc.) to navigate throughout the game.

-Use the "look at" command to describe any object mentioned in the game. Answers are often hidden in the descriptions.

-When in doubt, try to "take" any objects you encounter, as you may need them later on!

-Keep a pad and pencil on hand and try sketching your path through the game if you get lost.

COMMON COMMANDS:

TAKE - use this command to pick up a tool or objects. You may need the object later on!

INVENTORY / I - use this command to see the items you are carrying

DROP - use this command to stop carrying an item in your inventory

READ - use this command to read items you've taken or that you encounter

TALK - use this command to address a character in the game

LOOK AT - use this command to read a description of an object or area in more detail. Answers are often hidden in the descriptions.

LOOK - use this command to get a description of where you are

OPEN - use this command to open closed doors

UNLOCK - use this command when you have a key for doors that may be locked

TIE - use this command with rope or string

HINT - use this command if you get stuck in the adventure and need some guidance. More than one hint may be offered.
